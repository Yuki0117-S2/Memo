Windows 명령어 치트시트 — 명령어를 탭하면 바로 복사됨
파일 정리 · 백업 · 시스템 유틸 명령어 모음
전부 Windows 기본 내장 도구. PS 배지는 PowerShell, CMD 배지는 명령 프롬프트에서 실행. PowerShell에서는 CMD 명령도 대부분 그대로 동작한다.
RoboCopy 백업 치트시트
CMD
Windows 내장 복사 도구. 탐색기보다 안정적이고, 이어받기·미러링·선택 복사까지 지원한다.
기본 복사
robocopy "원본폴더" "목적지폴더" /E /Z /MT:16 ⧉ 복사
robocopy "원본폴더" "목적지폴더" /E /Z /MT:16 /R:3 /W:5 ⧉ 복사
실패 시 3번 재시도, 5초 간격. 실전에서 권장하는 기본 세트.
동기화 / 미러링
robocopy "원본" "백업" /MIR /Z /MT:16 ⧉ 복사
원본과 백업을 완전히 동기화. 원본에서 지운 파일은 백업에서도 삭제됨. 주의해서 사용.
robocopy "원본" "백업" /E /XO /Z /MT:16 ⧉ 복사
새 파일·변경 파일만 복사. 삭제는 따라가지 않아 LoRA·모델 백업에 안전.
선택적 복사
robocopy "원본" "목적지" *.safetensors /S /Z ⧉ 복사
특정 확장자 파일만 골라서 복사. safetensors, png 등 원하는 확장자로 변경 가능.
robocopy "원본" "목적지" /E /XD "제외할폴더명" ⧉ 복사
특정 폴더는 건너뛰고 복사. 여러 폴더 제외 시 폴더명을 공백으로 나열.
robocopy "E:\원본" "D:\목적지" /E /Z /MT:16 /XD "E:\원본\제외폴더" ⧉ 복사
전체 경로로 콕 집어 제외. 같은 이름 폴더가 여러 군데 있을 때 특정 하나만 정확히 빼고 싶으면 폴더명 대신 전체 경로를 적는다.
robocopy "원본" "목적지" /E /XF "제외할파일명" ⧉ 복사
특정 파일을 건너뛰고 복사. config.json처럼 파일명을 적거나 *.log처럼 와일드카드로 지정 가능.
로그 저장
robocopy "원본" "목적지" /E /Z /MT:16 /LOG:C:\robocopy_log.txt ⧉ 복사
복사 결과(성공·실패·스킵 파일 목록)를 txt 파일로 저장.
이동 / 미리보기
robocopy "원본" "목적지" /E /Z /MT:16 /MOVE ⧉ 복사
복사가 아니라 진짜 이동. 복사 완료 후 원본을 자동 삭제한다. 드라이브 정리·공간 확보에 유용하나, 원본이 사라지므로 신중히.
robocopy "원본" "목적지" /E /Z /MT:16 /L ⧉ 복사
실제 복사는 하지 않고 "이렇게 복사될 예정"이라는 결과만 미리 보여주는 시뮬레이션. 명령어가 맞는지, 어떤 파일이 옮겨질지 사고 전에 확인하는 용도.
TIP — 목적지 폴더는 미리 안 만들어도 됨. 목적지 경로의 폴더가 없으면 robocopy가 자동으로 만들어준다. D:\새폴더가 없어도 그냥 경로만 적으면 폴더부터 생성하고 내용을 채운다.
옵션 요약
  • /E — 하위 폴더 + 빈 폴더까지 전부 복사
  • /Z — 이어받기 모드. 중단된 파일 재시작 지원
  • /MT:16 — 16스레드 병렬 복사. SSD 환경에서 속도 향상
  • /MIR — 완전 미러링. 원본 삭제 = 백업 삭제이므로 주의
  • /XO — 원본보다 오래된 파일만 덮어씀 (새 파일·변경 파일만 추가)
  • /R:N /W:N — 재시도 횟수 N회, 대기 간격 N초 설정
  • /XD "폴더명" — 해당 폴더 제외 (전체 경로로 적으면 특정 하나만 정확히 제외)
  • /XF "파일명" — 해당 파일 제외 (*.log 같은 와일드카드 가능)
  • /MOVE — 복사 후 원본 삭제 (진짜 이동). 원본이 사라지므로 주의
  • /L — 미리보기 모드. 실제 복사 없이 결과만 시뮬레이션
파일 타임스탬프 변경
PS
탐색기의 "수정한 날짜"는 못 바꾸지만 PowerShell로는 자유자재. 만든 날짜 CreationTime · 수정한 날짜 LastWriteTime · 액세스한 날짜 LastAccessTime 세 가지가 있다.
파일 하나
(Get-Item "C:\경로\파일.png").LastWriteTime = "2026-06-27 02:55" ⧉ 복사
수정한 날짜를 원하는 시각으로 변경. 날짜만 적으면 00:00으로 설정됨.
폴더 안 전부 같은 시간으로
Get-ChildItem "C:\경로" | ForEach-Object { $_.LastWriteTime = "2026-06-27 02:55" } ⧉ 복사
하위 폴더까지 포함하려면 Get-ChildItem "C:\경로" -Recurse -File로 변경.
만든 날짜 + 수정한 날짜 세트로
$f = Get-Item "C:\경로\파일.png"; $f.CreationTime = "2026-06-27 02:55"; $f.LastWriteTime = "2026-06-27 02:55" ⧉ 복사
만든 날짜와 수정한 날짜를 한 번에 같은 시각으로 맞춘다.
폴더 날짜를 '지금'으로 (정렬 맨 위로 올리기)
(Get-Item "E:\경로\폴더이름").LastWriteTime = Get-Date ⧉ 복사
폴더도 파일과 똑같이 변경 가능. Get-Date는 실행하는 순간의 현재 시각 — 탐색기에서 "수정한 날짜" 정렬 시 맨 위로 올라온다. 지정 날짜로 하려면 Get-Date 대신 "2026-07-06 15:30"처럼 적기.
여러 폴더 한꺼번에 (오타 체크 포함)
$base = "E:\006_AIpicture\OriginLora"; "폴더A","폴더B","폴더C" | ForEach-Object { $p = Join-Path $base $_; if (Test-Path $p) { (Get-Item $p).LastWriteTime = Get-Date; Write-Host "완료: $_" -ForegroundColor Green } else { Write-Host "없음: $_" -ForegroundColor Red } } ⧉ 복사
$base에 상위 경로, 따옴표 목록에 폴더 이름들을 적는다. Test-Path가 실존 여부를 검사해서 이름이 틀리면 빨간색 "없음"으로 알려줌. 지정 날짜로 바꾸려면 여기서도 Get-Date 자리에 "2026-07-06 15:30"을 넣으면 된다. 폴더 자체의 날짜만 바뀌고 안에 든 파일들은 그대로.
실패하는 경우: 읽기 전용 파일, 권한 부족, 일부 클라우드 동기화 폴더(OneDrive 등)에서는 변경이 막힐 수 있다. GUI가 편하면 무료 툴 NewFileTime도 있음 (드래그 앤 드롭 일괄 변경).
일괄 리네임 & 폴더 정리
PS
연번 붙여 일괄 리네임
Get-ChildItem *.png | ForEach-Object -Begin {$i=1} -Process { Rename-Item $_ ("bianca_{0:D3}.png" -f $i++) } ⧉ 복사
bianca_001.png, bianca_002.png… 순서로 리네임. {0:D3}은 3자리 0채움 — 자릿수 바꾸려면 D3을 D4 등으로. Asset Renamer 열기 귀찮을 때 급하게 쓰는 용도.
확장자별로 폴더 만들어 자동 분류
Get-ChildItem -File | Group-Object Extension | ForEach-Object { $dir = $_.Name.TrimStart("."); mkdir $dir -Force | Out-Null; $_.Group | Move-Item -Destination $dir } ⧉ 복사
현재 폴더의 파일을 png / json / txt … 확장자명 폴더로 자동 분류해서 이동. 다운로드 폴더 대청소용.
폴더 용량 큰 순으로 확인
Get-ChildItem -Directory | ForEach-Object { [PSCustomObject]@{Name=$_.Name; MB=[math]::Round((Get-ChildItem $_ -Recurse -File | Measure-Object Length -Sum).Sum/1MB,1)} } | Sort-Object MB -Descending ⧉ 복사
하위 폴더별 용량을 MB 단위로 계산해 큰 순으로 정렬. 드라이브 어디가 터졌는지 찾을 때.
소소하지만 은근 쓰는 것들
CMD
폴더 구조를 텍스트로 뽑기
tree /F > 구조.txt ⧉ 복사
폴더 구조 전체를 트리 형태 텍스트로 저장. AI한테 프로젝트 구조 보여줄 때 복붙용으로 최고. 파일명 빼고 폴더만 보려면 /F 제거.
두 텍스트 파일 비교
fc 파일1.txt 파일2.txt ⧉ 복사
두 파일의 다른 부분을 출력. json 설정 파일 두 버전 뭐가 다른지 급하게 볼 때.
어떤 실행 파일이 잡히는지 경로 확인
where python ⧉ 복사
python 입력 시 실제로 어느 경로의 python이 실행되는지 전부 나열. ComfyUI 임베디드 파이썬 vs 시스템 파이썬 꼬였을 때 생명의 은인. PowerShell에서는 where.exe python으로 실행 (where만 치면 다른 명령으로 인식됨).
